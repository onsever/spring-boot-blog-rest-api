# Blog Application REST API
This is a blog application that allows users to create, read, update, and delete blog posts. It also allows users to create, read, update, and delete comments on blog posts.

## Project Features
* Java
* Maven
* MySQL
* Spring Boot
* Spring Data JPA
* Spring Web
* Spring Security
* Lombok
* ModelMapper
* JWT (JSON Web Token)
* Hibernate Validator
